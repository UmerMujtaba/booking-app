# Booking App

A React Native booking application built with Expo and Supabase.

## Prerequisites

- Node.js
- pnpm (or npm/yarn)

## Installation

1. Install dependencies:

```bash
pnpm install
```

## Running the app

Start the development server:

```bash
pnpm run dev
```

This will start the Expo development server. You can then open the app using the Expo Go app on your phone, or run it on an iOS/Android simulator.

### Other Commands

- `pnpm run build`: Build the project (using `scripts/build.js`).
- `pnpm run serve`: Serve the built project (using `server/serve.js`).
- `pnpm run typecheck`: Run TypeScript type checking.

## Tech Stack

- **Framework**: React Native + Expo
- **Routing**: Expo Router
- **Database/Backend**: Supabase
- **Language**: TypeScript
- **State Management**: TanStack React Query
- **Styling**: React Native Reanimated, Expo Vector Icons
- **i18n**: i18next + react-i18next
